# 🍽 Restaurant Analytics Dashboard

An interactive **Restaurant Analytics Dashboard** built using **Python, Streamlit, Pandas, and Plotly**.
This project analyzes restaurant data to generate insights related to cuisines, pricing, customer ratings, delivery trends, geographic patterns, and business growth opportunities.

---

## 📌 Project Overview

The objective of this project is to perform data analysis on a restaurant dataset and present insights through an interactive dashboard.

The dashboard is structured into multiple levels:

* **Level 1:** Basic Analysis
* **Level 2:** Intermediate Analysis
* **Level 3:** Advanced Analysis
* **Final Insights:** Executive Summary & Recommendations

---

## 🚀 Features

### 🏠 Home Dashboard

* Total Restaurants

* Top Cuisines

### 📈 Level 3 Analysis

### 💡 Final Insights

### 🎉 UI Enhancements

## 📊 Key Insights Generated

* A few cuisines dominate restaurant demand.
* Mid-priced restaurants attract the largest audience.
* Highly rated restaurants gain stronger trust and engagement.
* Online delivery improves customer convenience and reach.
* Top cities show higher restaurant competition and business activity.
* Chains benefit from consistency and brand loyalty.
